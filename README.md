## 📌 Overview

A time series machine learning project for forecasting energy demand. Uses historical consumption data and external features to train predictive models, enabling smarter energy planning and grid management.

**Tech Stack:** `Python` `Time Series` `Scikit-learn` `Pandas` `Matplotlib` `Forecasting`

---

## 🚀 Getting Started

### Prerequisites

```bash
pip install -r requirements.txt
```

> Requires **Python 3.8+**

### Installation

```bash
git clone https://github.com/Onabanjomicheal/Energy-Demand-Forecasting.git
cd Energy-Demand-Forecasting
pip install -r requirements.txt
```

### Run

```bash
python main.py
```
